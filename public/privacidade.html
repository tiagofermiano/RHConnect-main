Política de Privacidade

Esta Política de Privacidade descreve como o sistema acadêmico RHConnectCV coleta, utiliza e armazena os dados pessoais dos candidatos que utilizam o módulo de cadastro de contas e currículos.

1. Dados coletados

Podemos coletar os seguintes dados pessoais:

  • Nome completo;
  • E-mail;
  • Telefone;
  • Informações sobre a vaga de interesse;
  • Link para o currículo (como LinkedIn, Drive ou arquivo PDF);
  • Registros de consentimento (data e hora em que o usuário aceitou os termos).

2. Finalidade do uso dos dados

Os dados são utilizados exclusivamente para:

  • Permitir a criação e autenticação da conta do candidato;
  • Salvar e exibir os currículos cadastrados pelo próprio candidato;
  • Possibilitar contato futuro sobre oportunidades compatíveis com o perfil cadastrado.

3. Base legal e consentimento

O tratamento de dados neste sistema é baseado no consentimento do titular. O usuário só consegue criar sua conta e cadastrar currículos após marcar a caixa de “Li e aceito”, concordando com esta Política de Privacidade.

4. Compartilhamento de dados

Este sistema acadêmico não compartilha os dados com terceiros para fins de marketing ou publicidade. Os dados podem ser acessados apenas pelo próprio candidato e, em cenários reais, por equipes de recrutamento autorizadas.

5. Armazenamento e segurança

Os dados são armazenados em um banco de dados relacional (MySQL) e o acesso é protegido por autenticação via usuário e senha. Em uma implantação real, recomenda-se o uso de criptografia, controles de acesso mais rígidos e auditoria de acessos.

6. Direitos do titular

O candidato pode, a qualquer momento:

  • Atualizar seus dados de perfil na área autenticada;
  • Solicitar a exclusão da conta e dos currículos cadastrados (neste protótipo, essa ação pode ser simulada em aula);
  • Revogar o consentimento, interrompendo o uso do sistema.

7. Cookies

Utilizamos cookies apenas para manter a sessão ativa e melhorar a experiência de navegação. O banner exibido ao acessar o sistema informa o usuário e permite o aceite explícito.

8. Ambiente acadêmico

Este sistema foi desenvolvido para fins exclusivamente acadêmicos, como exemplo de boas práticas de desenvolvimento web aliado a princípios de ética e proteção de dados pessoais.

Em caso de dúvidas sobre esta Política de Privacidade, recomenda-se discutir com o professor responsável pela disciplina ou equipe de desenvolvimento.
